# Konisoft Speedruns

A **Konisoft Speedruns** egy dedikált webes platform, ahol a játékosok mérhetik fel ügyességüket és gyorsaságukat a Konisoft által kiadott játékokban. Legyen szó egy teljes végigjátszásról, egy trükkös pályarészről vagy egy  fejezetről!

---

## 🚀 Főbb Funkciók

* **Globális Ranglisták:** Kövesd nyomon, ki a leggyorsabb a különböző kategóriákban.
* **Játék Specifikus Bontás:** Külön leaderboardok minden Konisoft játékhoz.
* **Részletes Felosztás:** Mérd az idődet pályákra, fejezetekre vagy teljes játékmenetre lebontva.
